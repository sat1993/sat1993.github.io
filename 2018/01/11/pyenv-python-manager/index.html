pyenv是什么？

centos7默认的python为2.7.9版本。而neovim需要3以上版本的python。
而python的卸载、覆盖安装都是很麻烦，并且容易出错。可以通过pyenv简单搞定多版本管理。

pyenv是python版本管理工具，可以快速在命令行切换不同版本的python。
