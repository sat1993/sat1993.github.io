docker简介

docker是一个容器，将服务封装到docker image中，以container的形式启动。
主要可以做到服务间的隔离，保证不会出现相互侵蚀的情况。

使用Docker有以下好处：

  1. 更高效的利用系统资源
  2. 更快速的启动时间
  3. 一致的运行环境
  4. 持续交付和部署
  5. 更轻松的迁移
  6. 更轻松的维护和扩展

对个人开发者来说，docker最明显的好处就是能快速搭建开发所需要的环境(kafka、mysql、redis等)。
摆脱手动安装的困扰，需要的话docker pull一下，再不济build一个新镜像。
